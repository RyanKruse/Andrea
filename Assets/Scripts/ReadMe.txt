			  Introduction

Andrea is a virtual reality reader.

Enables reading large amounts of text in VRChat.

Examples include:

	1) Books.
	2) Stories.
	3) Articles.
	4) Essays.
	5) Tutorials.
	6) Blogs.
	7) Theater.
	8) Stand-Up.
	9) Speeches.
	10) Lyrics.

----------------------------------------

				Tutorial

Andrea works in the following execution order:

	1) Populate the text files.
	2) Format the text files.
	3) Process the text files.
	4) Compile the text files.
	5) Read the text files.
